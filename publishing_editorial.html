Print is permanent - once it's on paper, there's no undo button. Master these workflows, and your work reaches audiences worldwide!

Technical Specifications by Format

Standard Publishing Formats

Children's Picture Books:
────────────────────────────────────
Trim Size: 8" × 10" or 8.5" × 11" (common)
Resolution: 300-350 DPI minimum
Color Space: CMYK for print
Bleed: 0.125" (1/8") all sides
Page Count: 32 pages typical (must be multiple of 8)
File Format: Layered PSD or TIFF, final PDF
Spread Work: Design as facing pages

Young Adult/Middle Grade Novels:
────────────────────────────────────
Trim Size: 5.5" × 8.5" or 6" × 9"
Interior: Black & white or grayscale at 600 DPI
Cover: Full color, 300 DPI, CMYK
Spine Width: Depends on page count and paper
File Format: PDF/X-1a or PDF/X-4

Graphic Novels/Trade Paperbacks:
────────────────────────────────────
Trim Size: 6" × 9" (common) or 6.625" × 10.25"
Resolution: 300-400 DPI
Color Space: CMYK (print) or RGB (digital)
Bleed: 0.125" to 0.25"
Page Count: Multiple of 4 (or 8 for perfect binding)
File Format: High-res PDF with embedded fonts

Comics (Floppy/Single Issue):
────────────────────────────────────
Trim Size: 6.625" × 10.25" (standard)
Print Area: 6" × 9.25" (live area)
Full Bleed Area: 6.875" × 10.5"
Resolution: 300-400 DPI for color, 600 DPI for line art
Color Space: CMYK
File Format: TIFF or PSD layers, final PDF

Magazine Editorial:
────────────────────────────────────
Size: Varies (8.5" × 11" common)
Resolution: 300 DPI minimum
Color Space: CMYK for print, RGB for digital
Format: TIFF, PSD, or high-res JPEG
Deadline: Usually 1-2 weeks (FAST turnaround)

Digital/E-book:
────────────────────────────────────
Format: Fixed-layout (images) or reflowable (text)
Resolution: 150-300 DPI (lower for file size)
Color Space: RGB (screen display)
Aspect Ratio: Common 4:3 or 16:9
File Size: Optimize for download (< 5MB ideal)
Format: EPUB, MOBI, or PDF

Color Management for Print

🎨 RGB vs CMYK - Critical Understanding

The Fundamental Difference:
────────────────────────────────────

RGB (Screen):
• Red, Green, Blue light
• Additive color (light adds together)
• Wider color gamut
• What you paint in
• 16.7 million colors

CMYK (Print):
• Cyan, Magenta, Yellow, Black ink
• Subtractive color (ink absorbs light)
• Smaller color gamut
• What prints
• Fewer colors possible

The Problem:
Many RGB colors cannot be reproduced in CMYK!
Bright blues, oranges, greens especially

Workflow Solution:
────────────────────────────────────

Option 1: Paint in RGB, Convert to CMYK
1. Paint in RGB (Paintstorm default)
2. Soft proof with CMYK preview
3. Adjust colors that are out of gamut
4. Convert to CMYK for final delivery
5. Check again after conversion

Option 2: Paint Directly in CMYK
1. Set Paintstorm to CMYK mode
2. Colors look duller (that's correct!)
3. What you see is what prints
4. No surprises at print

Recommendation:
• Illustrations: Paint in RGB, convert carefully
• Comics: Consider working in CMYK from start
• Editorial: RGB usually OK, convert at end

Soft Proofing:
────────────────────────────────────
View > Proof Setup > Working CMYK
(In Photoshop or similar)

Shows how colors will print
Adjust before converting
Critical step!

Common CMYK Issues:
────────────────────────────────────
❌ Bright electric blue → Duller in CMYK
❌ Vibrant orange → More red-orange in CMYK  
❌ Pure RGB green → Yellower in CMYK
❌ Saturated purple → Less vibrant in CMYK

Solution:
✓ Stay within CMYK-safe colors when possible
✓ Adjust expectations
✓ Use Pantone spot colors for critical brand colors
✓ Test print early!

File Preparation Checklist

Pre-Press Requirements

Before Submitting to Publisher:
────────────────────────────────────

☐ Resolution Check
  • 300 DPI minimum for color
  • 600 DPI for black & white line art
  • Verify at actual print size

☐ Color Mode
  • Convert RGB to CMYK if required
  • Check for out-of-gamut colors
  • Total ink coverage under 300% (240-280% ideal)

☐ Bleeds and Margins
  • Bleed: Usually 0.125" beyond trim
  • Safe area: 0.25" inside trim (for critical content)
  • Nothing important in bleed area

☐ Layers and Transparency
  • Flatten if required (check with printer)
  • OR keep layers but test flattening
  • Ensure transparency flattens correctly

☐ Text and Fonts
  • Fonts embedded or outlined
  • Text in separate layer if editable needed
  • Check for font licensing

☐ File Format
  • PSD/TIFF with layers for working files
  • PDF/X-1a or PDF/X-4 for final print
  • High-res JPEG acceptable for some uses

☐ File Naming
  • Clear, descriptive names
  • Include page number for multi-page
  • Version number
  • Example: BookTitle_Page05_v03_FINAL.pdf

☐ Color Profiles
  • Embed ICC color profile
  • Use printer's preferred profile if provided
  • Common: US Web Coated (SWOP) v2

☐ Quality Check
  • Print test on desktop printer
  • Check for artifacts or issues
  • Verify all elements present
  • Spell check text elements

☐ Backup
  • Keep layered working files
  • Save at multiple stages
  • External backup critical

Book Illustration Workflows 📕

Book illustration is a storytelling art form. Whether creating children's picture books, chapter book illustrations, or cover art, you're enhancing narrative, establishing mood, and inviting readers into worlds. Every illustration must serve the story!

Children's Picture Book Workflow

Complete Production Process

Stage 1: Manuscript Analysis (1-2 days)
────────────────────────────────────

Read Manuscript Multiple Times:
• First read: Overall story feel
• Second read: Mark potential illustrations
• Third read: Note emotional beats

Create Scene Breakdown:
• List all scenes/moments
• Identify key story beats
• Note character appearances
• Mark scene changes

Typical 32-Page Picture Book:
• 14-16 double-page spreads
• Not every spread needs full illustration
• Some text-only pages for pacing
• Climax needs strongest illustration

Questions to Answer:
• What's the emotional arc?
• Which moments MUST be shown?
• Where does action happen?
• What can be implied vs shown?
• Character consistency needs?

Stage 2: Thumbnail Storyboard (3-5 days)
────────────────────────────────────

Create Rough Layout:
• Sketch all spreads at small size
• Just shapes and composition
• Show where text goes
• Establish pacing and flow

Consider:
• Page turns (surprise, reveal)
• Spread vs single page
• Text placement
• Focal points
• Character entry/exit
• Scene transitions

Review with Editor:
• Get feedback on pacing
• Adjust scene selection
• Refine before detail work

Stage 3: Character Design (3-5 days)
────────────────────────────────────

Design Main Characters:
• Multiple exploration sketches
• Turnarounds for consistency
• Expression sheet
• Color palette

Design Supporting Elements:
• Locations
• Props
• Background characters

Create Style Guide:
• For your own consistency
• Character proportions
• Color palette
• Visual style notes

Stage 4: Sketch Phase (2-3 weeks)
────────────────────────────────────

Create Detailed Sketches:
• Full-size (or close) sketches
• Show composition clearly
• Character poses refined
• Background elements placed
• Text integration shown

Per Spread:
• Draw at print size or larger
• Gray value to show depth
• Clear enough for approval
• Don't overwork (it's still sketch!)

Submit for Approval:
• Usually all spreads at once
• OR in batches
• Incorporate feedback
• May need 1-2 revision rounds

Stage 5: Final Art (4-8 weeks)
────────────────────────────────────

Painting Process Per Spread:
1. Transfer sketch to final canvas
2. Block in base colors
3. Develop forms and values
4. Add details and textures
5. Final polish and effects
6. Review and adjust

Time per Spread:
• Simple: 1-2 days
• Medium: 2-4 days
• Complex: 4-7 days

Working Method:
• Some do all spreads together
• Others finish one spread at a time
• Benefits to both approaches

Stage 6: Text Integration (1-2 weeks)
────────────────────────────────────

Collaborate with Designer:
• Provide art with clear text areas
• OR designer places text on art
• Adjust art if text doesn't fit
• Balance text and image

Consider:
• Reading order
• Text legibility on backgrounds
• Speech attribution
• Text color/style

Stage 7: Final Delivery (1 week)
────────────────────────────────────

Prepare Files:
• Convert to CMYK if needed
• Add bleeds
• Proper resolution (300 DPI)
• Correct file format
• Name files clearly

Include:
• All final spreads
• Cover if contracted
• Any spot illustrations
• Working files as backup

Quality Check:
• Print test copies
• Check all pages present
• Verify nothing cut off
• Ensure text areas clear

Picture Book Art Styles

🎨 Choosing the Right Style

Consider Your Story:

Realistic/Representational:
────────────────────────────────────
Best For:
• Historical fiction
• Nature/science books
• Serious topics
• Older picture book readers (8-10)

Characteristics:
• Accurate proportions
• Realistic colors and lighting
• Detailed environments
• Believable characters

Examples: Jerry Pinkney, Brian Selznick

Stylized/Simplified:
────────────────────────────────────
Best For:
• Younger readers (3-6)
• Humorous stories
• Abstract concepts
• Modern stories

Characteristics:
• Simplified shapes
• Bold colors
• Clear silhouettes
• Easy to "read"

Examples: Oliver Jeffers, Jon Klassen

Painterly/Impressionistic:
────────────────────────────────────
Best For:
• Poetic stories
• Emotional narratives
• Atmospheric tales
• Literary picture books

Characteristics:
• Visible brushwork
• Color and light emphasis
• Mood over precision
• Artistic interpretation

Examples: Shaun Tan, Chris Van Allsburg

Cartoon/Graphic:
────────────────────────────────────
Best For:
• Funny stories
• Action-packed tales
• Character-driven plots
• Series books

Characteristics:
• Expressive characters
• Clear lines
• Exaggerated features
• Easy to animate mentally

Examples: Mo Willems, Dav Pilkey

Textured/Mixed Media Look:
────────────────────────────────────
Best For:
• Unique visions
• Tactile stories
• Artistic statements
• Standout aesthetic

Characteristics:
• Textural variety
• Digital + traditional feel
• Rich surface quality
• Distinctive look

Examples: Eric Carle (collage), Melissa Sweet

Matching Style to Story:
────────────────────────────────────

Funny Story → Cartoon/exaggerated
Scary Story → Dark, dramatic shadows
Bedtime Story → Soft, soothing colors
Adventure Story → Dynamic compositions
Quiet Story → Intimate, detailed scenes
Fantasy Story → Imaginative, atmospheric

Composition Considerations for Picture Books

Spread vs Single Page Design

Double-Page Spread:
────────────────────────────────────
When to Use:
✓ Important story moments
✓ Establishing shots (wide scenes)
✓ Dramatic reveals
✓ Action sequences
✓ Emotional climaxes

Design Considerations:
• Nothing critical in gutter (center fold)
• Composition flows across both pages
• Characters can span the gutter carefully
• Balance across spread, not individual pages

Single Page Focus:
────────────────────────────────────
When to Use:
✓ Intimate character moments
✓ Dialogue-heavy scenes
✓ Quick pacing
✓ Alternating with text pages

Design Considerations:
• Self-contained composition
• Focal point on one side
• Can be left or right page
• Balance with facing text

Text Placement Strategies:
────────────────────────────────────

Top of Spread:
• Text above, image below
• Image can be large
• Clear reading order

Bottom of Spread:
• Image above, text below
• Dramatic image showcase
• Text becomes caption

Integrated in Image:
• Text within scene
• Most immersive
• Careful color/contrast needed
• Common in modern books

Separate Text Page:
• Text on left, image on right (or vice versa)
• Traditional approach
• Clear separation
• Easy to read
📚 Picture Book Wisdom: "The best picture books show what words cannot tell, and tell what pictures cannot show. Text and image dance together, each enhancing the other. Never simply illustrate the words - extend and enrich them!"

Comic Art Production 💥

Comic production combines art and storytelling in a unique way. Whether creating graphic novels, webcomics, or traditional comic books, you're crafting visual narrative through sequential panels. Every page is a carefully orchestrated reading experience!

Comic Page Workflow

📖 Complete Comic Page Process

Stage 1: Script Breakdown (30 mins per page)

Analyze the Script:
• How many panels?
• What happens in each panel?
• What dialogue/captions?
• What's the emotional beat?
• Any specific shot requirements?

Example Script:
────────────────────────────────────
PAGE 5 - 5 PANELS

Panel 1: Wide shot. Hero stands on rooftop, city behind.
HERO: "This ends tonight."

Panel 2: Close-up of villain's face, grinning.
VILLAIN: "Does it?"

Panel 3: Medium shot. Hero leaps from rooftop.
(No dialogue)

Panel 4: Action. Hero and Villain clash mid-air.
SFX: KRAKOOM!

Panel 5: Impact. Both crash into billboard.
HERO: "Ugh!"

Your Job: Visualize this effectively

Stage 2: Thumbnails (1-2 hours per page)

Rough Layout Planning:
• Draw page at 2-3 inches tall
• Sketch panel borders
• Indicate figures (stick figures OK)
• Note camera angles
• Mark text areas roughly

Goals:
✓ Readable flow
✓ Clear storytelling
✓ Dynamic composition
✓ Varied panel sizes
✓ Rhythm and pacing

Don't worry about:
✗ Details
✗ Anatomy
✗ Final composition
✗ Specific poses

Stage 3: Layouts/Pencils (4-8 hours per page)

Full-Size Drawing:
• Work at 6.875" × 10.5" (with bleed)
• Or larger (11" × 17" traditional)
• Define all panels clearly
• Draw figures accurately
• Show perspective and backgrounds
• Indicate blacks and shadows

Level of Detail:
• If you're inking: moderate detail
• If you're finishing: full detail
• If sending to inker: very clear

Common Approach:
• Light blue pencil for construction
• Tight pencils for finals
• Or all digital with layers

Stage 4: Inking (3-6 hours per page)

Finalizing Line Art:
• Clean, confident lines
• Vary line weight
• Thick for important/close
• Thin for distant/less important
• Spot blacks (solid black areas)

Digital Inking in Paintstorm:
• Use pressure-sensitive brush
• Steady, smooth strokes
• Undo is OK, but commit to lines
• Separate layer from pencils
• Can work at high resolution (600 DPI)

If Sending to Colorist:
• Clean lines essential
• Close all shapes
• No stray marks
• Export as high-contrast B&W

Stage 5: Coloring (4-8 hours per page)

Color Workflow:
1. Flatting (base colors)
   • Fill each area with flat color
   • Stay within lines
   • Separate layers per element
   • Can be done by assistant

2. Rendering (adding depth)
   • Add shadows
   • Add highlights  
   • Show form and volume
   • Establish lighting

3. Effects
   • Special effects (fire, energy)
   • Atmospheric elements
   • Glows and lighting

4. Final Adjustments
   • Color correction
   • Consistency across pages
   • Mood and atmosphere

Stage 6: Lettering (1-2 hours per page)

Adding Text:
• Usually done in separate program (Illustrator)
• Or in Photoshop/Paintstorm
• Clear, legible font
• Proper reading order
• Balloon placement careful

Lettering Guidelines:
• Word balloons don't obscure important art
• Tail points to speaker clearly
• Reading order flows naturally (left-right, top-bottom)
• Font size appropriate (usually 10-12pt)
• SFX integrated into art

Final File:
• Combine all layers
• Check resolution (300 DPI color, 600 DPI B&W)
• Convert to CMYK if printing
• Add bleeds
• Export as TIFF or PDF

Comic Panel Composition

Panel Layout Principles

Panel Size = Importance + Time:
────────────────────────────────────

Large Panel:
• Important moments
• Longer time duration
• Demands attention
• Use sparingly for impact

Small Panel:
• Quick moments
• Rapid time passage
• Less important actions
• Creates rhythm

Equal Panels:
• Steady rhythm
• Regular pacing
• Methodical feel
• Classic grid

Panel Shape Psychology:
────────────────────────────────────

Horizontal (Landscape):
• Wide shots
• Establishing scenes
• Calm, stable feeling
• Passage of time

Vertical (Portrait):
• Tall buildings, trees
• Characters standing
• Dramatic, powerful
• Emphasis on height/depth

Square:
• Neutral, balanced
• Consistent pacing
• Grid comics
• Clean, modern

Irregular:
• Dynamic, energetic
• Breaks conventions
• Jarring or exciting
• Action sequences

Reading Flow:
────────────────────────────────────

Western Comics (Left to Right):
Row 1: [1] [2] [3]
Row 2: [4] [5] [6]
Row 3: [7] [8] [9]

Manga (Right to Left):
Row 1: [3] [2] [1]
Row 2: [6] [5] [4]
Row 3: [9] [8] [7]

Confusing Flow Example:
[1]     [3]
    [2]
[4] [5] [6]

✗ Ambiguous order!
✓ Use clear z-pattern or grid

Gutter Space:
────────────────────────────────────
The space between panels matters!

Tight Gutters (1-2mm):
• Rapid pacing
• Connected moments
• Intense action

Wide Gutters (5-10mm):
• Slower pacing
• Distinct moments
• Breathing room

No Gutters (Panels Touch):
• Extreme speed
• Simultaneous action
• Visual impact

Comic Storytelling Techniques

Visual Narrative Tools

Camera Angles:
────────────────────────────────────

Eye Level:
• Neutral viewpoint
• Reader = observer
• Most common
• Informational

High Angle (Looking Down):
• Character appears small, weak
• Viewer has power
• Vulnerable feeling

Low Angle (Looking Up):
• Character appears large, powerful
• Viewer feels small
• Intimidating, heroic

Dutch Angle (Tilted):
• Unease, tension
• Something wrong
• Disorientation
• Horror, suspense

Shot Distances:
────────────────────────────────────

Extreme Wide Shot:
• Entire environment
• Establishing location
• Character very small
• Scale and scope

Wide Shot:
• Character full body + environment
• Action visibility
• Spatial relationships

Medium Shot:
• Character waist-up
• Dialogue scenes
• Character interaction

Close-Up:
• Character head + shoulders
• Emotion focus
• Intimacy

Extreme Close-Up:
• Eyes, mouth, hand
• Intense emotion
• Important detail
• Dramatic impact

Transitions Between Panels:
────────────────────────────────────

Moment-to-Moment:
Panel 1: Character reaches for door
Panel 2: Character's hand on doorknob
• Very little time passes
• Slow, deliberate pacing

Action-to-Action:
Panel 1: Character winds up punch
Panel 2: Fist connects with target
• Single action progresses
• Clear causality

Subject-to-Subject:
Panel 1: Hero speaks
Panel 2: Villain responds
• Same scene, different subjects
• Most common
• Dialogue/interaction

Scene-to-Scene:
Panel 1: Daytime, office
Panel 2: Night, home
• Different time/location
• Scene change
• Story progression

Aspect-to-Aspect:
Panel 1: Clock showing time
Panel 2: Rain on window
Panel 3: Character's worried face
• Different aspects of same moment
• Mood building
• Atmospheric

Non-Sequitur:
Panel 1: Modern city
Panel 2: Prehistoric era
• No logical connection
• Surreal, dreamlike
• Artistic choice
💥 Comic Artist Truth: "Comics are not illustrated prose - they're a unique language combining words and images. Master the grammar of panels, gutters, and visual flow, and you can tell stories impossible in any other medium!"

Editorial Illustration 📰

Editorial illustration is idea visualization under pressure. You have days (sometimes hours!) to create artwork that captures complex concepts, complements articles, and catches readers' eyes. This is illustration as journalism - fast, smart, and impactful!

Editorial Workflow

Fast-Turnaround Process

Timeline: Typical 1-Week Assignment
────────────────────────────────────

Day 1: Assignment + Concept (3-4 hours)
• Receive article or topic
• Read thoroughly, take notes
• Identify key concepts/themes
• Sketch 3-5 different concepts
• Submit concepts to Art Director

Day 2: Feedback + Refinement (2-3 hours)
• Get concept feedback
• Make adjustments if needed
• Refine chosen concept
• Create rough for approval
• Get final approval

Days 3-4: Final Illustration (8-16 hours)
• Create final artwork
• Work efficiently but well
• Stay true to approved concept
• High quality essential

Day 5: Revisions + Delivery (2-4 hours)
• Minor tweaks if requested
• Final color adjustments
• Prepare file for print
• Deliver on deadline

CRITICAL: Deadline is ABSOLUTE
• Magazines go to press on schedule
• Miss deadline = problem
• Budget time wisely

Rush Jobs (24-48 hours):
────────────────────────────────────
Sometimes you get emergency assignments:

Hour 1-2: Concept (FAST)
• Read article quickly
• Sketch 2 concepts maximum
• Email immediately

Hour 3-4: Get approval, start final
• Art Director chooses
• Begin final art immediately

Hour 8-12: Finish
• Work through if needed
• Deliver on time

Rush Strategies:
✓ Simple, bold concepts
✓ Limit color palette
✓ Use proven techniques
✓ Don't experiment
✓ Focus on idea over finish

Editorial Concept Development

💡 Conceptual Thinking

Types of Editorial Illustration:
────────────────────────────────────

1. Literal:
   • Shows exactly what article describes
   • Clear, straightforward
   • Less creative but effective
   • Example: Article about gardening → person gardening

2. Metaphorical:
   • Uses metaphor/symbolism
   • More interpretive
   • Requires thought
   • Example: Article about economic growth → plant sprouting from coins

3. Conceptual/Abstract:
   • Visualizes ideas, not things
   • Most creative
   • Can be risky
   • Example: Article about connection → abstract web of lines

4. Humorous:
   • Funny take on topic
   • Engaging, memorable
   • Requires wit
   • Example: Article about multitasking → octopus juggling

Concept Development Process:
────────────────────────────────────

Step 1: Identify the Core Idea
Ask: "What is this article REALLY about?"
• Not the surface topic
• The underlying concept
• The emotional core

Example Article: "Social Media Addiction"
Surface: People use phones too much
Core Idea: Technology controls us / Loss of real connection

Step 2: List Visual Metaphors
Brainstorm images for the core idea:
• Control → Strings, puppet, cage, leash
• Addiction → Chain, trap, dependency
• Lost connection → Distance, walls, isolation
• Technology → Screens, wires, robots

Step 3: Combine and Refine
Mix metaphors in interesting ways:
• Phone as ball and chain
• Person trapped in phone screen
• Puppeteer phone controlling person
• People as phone zombies

Step 4: Choose Most Effective
Consider:
✓ Is it clear at a glance?
✓ Does it capture the idea?
✓ Is it fresh/not cliché?
✓ Will it reproduce well?
✓ Does it fit the tone?

Common Pitfalls:
────────────────────────────────────

❌ Too Literal
Article about rise in coffee sales
Bad: Just picture of coffee cup
Better: Coffee cup as rising sun

❌ Too Obscure
Concept so abstract no one gets it
Bad: Random shapes only artist understands
Better: Clear metaphor with style

❌ Cliché
Overused visual metaphor
Bad: Lightbulb for ideas (done to death)
Better: Fresh take on idea generation

❌ Doesn't Fit Article
Beautiful art that misses point
Bad: Pretty but irrelevant imagery
Better: On-topic even if simple

Strong Editorial Concepts:
────────────────────────────────────
✓ Understood in 3 seconds or less
✓ Works in black & white (test!)
✓ Fresh perspective on topic
✓ Appropriate tone for publication
✓ Scalable (works large or small)
✓ Memorable imagery

Print Technical Requirements 🖨️

Understanding print production isn't optional - it's essential. Beautiful digital art that doesn't print correctly is worthless to publishers. Master these technical requirements, and you'll deliver work that reproduces perfectly every time!

Resolution and Size Requirements

DPI/PPI Deep Dive

Understanding Resolution:
────────────────────────────────────

DPI = Dots Per Inch (printer)
PPI = Pixels Per Inch (screen)

For Print, Use DPI:
• 300 DPI = Standard for color printing
• 350 DPI = High-quality printing
• 600 DPI = Line art/black & white
• 1200 DPI = Ultra-fine detail (rare)

Why 300 DPI?
• Human eye can't distinguish more at normal viewing distance
• Smaller than 300 = visible pixelation
• Larger than 300 = unnecessarily huge files

Calculating Print Size:
────────────────────────────────────

Formula: Pixel Dimensions ÷ DPI = Print Size

Example 1:
3000 pixels wide ÷ 300 DPI = 10 inches wide
2000 pixels tall ÷ 300 DPI = 6.67 inches tall
Result: Can print at 10" × 6.67" at 300 DPI

Example 2:
Book needs to be 8" × 10"
8" × 300 DPI = 2400 pixels wide
10" × 300 DPI = 3000 pixels tall
Result: Create file at 2400 × 3000 pixels

Adding Bleeds:
────────────────────────────────────

Bleed = Image extends beyond trim
Standard: 0.125" (1/8 inch) all sides

8" × 10" book with bleed:
Width: 8" + 0.25" (0.125" × 2 sides) = 8.25"
Height: 10" + 0.25" = 10.25"

At 300 DPI:
8.25" × 300 = 2475 pixels wide
10.25" × 300 = 3075 pixels tall

Final canvas: 2475 × 3075 pixels

Safe Area (Margin):
────────────────────────────────────

Keep critical content inside safe area
Margin: Usually 0.25" from trim

Text should be:
• At least 0.25" from trim edge
• At least 0.375" from bleed edge

Why?
• Trim can be slightly off
• Text at edge gets cut off
• Looks unprofessional

Working Canvas Setup:
────────────────────────────────────

For 8" × 10" book spread (two pages):

Total width: 16" (two 8" pages)
Height: 10"
Add bleeds: 16.25" × 10.25"
At 300 DPI: 4875 × 3075 pixels

Guides to add:
• Bleed line (outer edge)
• Trim line (where it will be cut)
• Safe area (0.25" inside trim)
• Center gutter (between pages)

CMYK Color Management

🎨 Professional Color Workflow

RGB to CMYK Conversion:
────────────────────────────────────

In Paintstorm/Photoshop:
1. Image > Mode > CMYK Color
   OR
   Edit > Convert to Profile > CMYK

2. Choose CMYK Profile:
   • US Web Coated (SWOP) v2 (common)
   • Coated FOGRA39 (Europe)
   • Japan Color 2001 Coated
   • Ask your printer for their profile!

3. Conversion Options:
   • Engine: Adobe (ACE)
   • Intent: Relative Colorimetric or Perceptual
   • Use Black Point Compensation: YES
   • Use Dither: NO (for print)

What Happens During Conversion:
────────────────────────────────────

Some colors shift:
• Bright blues become duller
• Neon colors impossible in CMYK
• Vibrant greens shift toward yellow
• Pure RGB colors have no CMYK equivalent

Out of Gamut Colors:
────────────────────────────────────

View > Gamut Warning (Photoshop)
Shows which colors can't print

Fix Out of Gamut:
• Desaturate slightly
• Shift hue to printable version
• Accept the limitation

Prevention:
✓ Soft proof frequently during painting
✓ View > Proof Colors
✓ See how it will print while working
✓ Adjust colors before final

Total Ink Coverage:
────────────────────────────────────

CMYK values are percentages (0-100%)
Total Ink = C% + M% + Y% + K%

Maximum: Usually 300% total
Ideal: 240-280% for most printing

Example Problem:
C: 100%
M: 100%
Y: 100%
K: 100%
Total: 400% (TOO MUCH!)

Result: Ink bleeds, doesn't dry, mess

Fix:
Reduce black: K: 80%
Reduce other colors slightly
Total: 280% (Good!)

Rich Black vs Pure Black:
────────────────────────────────────

Pure Black (K: 100%):
• Looks gray on press
• Not deep enough
• Amateur mistake

Rich Black (Better):
C: 40%
M: 30%
Y: 30%
K: 100%
Total: 200%
Result: Deep, rich black

Text Exception:
Small text: Use pure K: 100% only
Why? Registration (alignment) issues
Other colors might misalign = blurry text

File Formats and Export

Choosing the Right Format

Working Files (Keep for Yourself):
────────────────────────────────────

PSD (Photoshop Document):
✓ Preserves layers
✓ Paintstorm can save as PSD
✓ Industry standard
✓ Editable later
Size: Large

TIFF (Tagged Image File Format):
✓ Lossless compression
✓ Can preserve layers
✓ High quality
✓ Widely compatible
Size: Very large

Delivery Files (Send to Publisher):
────────────────────────────────────

PDF (Portable Document Format):
✓ Industry standard for print
✓ Embeds fonts
✓ Preserves color profiles
✓ Can contain multiple pages
✓ PDF/X-1a or PDF/X-4 for print
Size: Medium

Settings for Print PDF:
• Compression: Automatic (JPEG)
• Quality: Maximum
• Color Conversion: No conversion (already CMYK)
• Include Bleed Marks: YES
• Include Crop Marks: Optional
• Embed all fonts: YES

TIFF (High-Res Final):
✓ For photo-like illustrations
✓ When PDF not accepted
✓ Lossless quality
Settings: LZW compression, 300 DPI

What NOT to Send:
────────────────────────────────────

❌ JPEG (lossy compression, quality loss)
❌ PNG (not for print)
❌ Low-resolution files
❌ RGB files when CMYK required
❌ Files without bleeds
❌ Files with wrong dimensions

File Checklist Before Delivery:
────────────────────────────────────

☐ Correct dimensions (with bleeds)
☐ 300 DPI resolution minimum
☐ CMYK color mode (if printing)
☐ All layers flattened (unless requested)
☐ No spot colors (unless specified)
☐ Embedded fonts (if text present)
☐ Color profile embedded
☐ File named correctly
☐ Test printed on desktop printer
☐ Visually inspected for errors
🖨️ Print Production Truth: "Digital looks forgiving - you can always adjust. Print is permanent. One wrong setting, and thousands of books print incorrectly. Master the technical, and you'll never fear the press!"

Sequential Storytelling 📖

Sequential art - whether comics or picture books - requires understanding how images work together to tell stories. It's not just individual pictures, it's the flow, rhythm, and connections between them that create narrative!

Pacing and Rhythm

Controlling Time Through Images

Visual Pacing Techniques:
────────────────────────────────────

Slow Pacing:
• Many panels showing small progression
• Detailed backgrounds
• Contemplative moments
• Emotional beats
• Reader takes time

Example: Character walking through forest
Panel 1: Wide shot, character small
Panel 2: Medium shot, walking
Panel 3: Close on feet stepping
Panel 4: Close on face, thoughtful
Panel 5: Hand touching tree
Panel 6: Wide shot, deeper in forest
Time: 30 seconds shown in 6 panels = slow

Fast Pacing:
• Few panels covering much action
• Simplified backgrounds
• Action-focused
• Excitement, urgency
• Quick reading

Example: Chase scene
Panel 1: Hero runs from explosion
Panel 2: Leaps over obstacle
Panel 3: Crashes through window
Time: 3 seconds shown in 3 panels = fast

Using Panel Count:
────────────────────────────────────

One Panel Per Page Turn:
• Maximum dramatic impact
• Reveals with page turn
• Shocking moments
• Climactic scenes

Full Page Splash:
• Wordless impact
• Iconic moment
• Action peak
• "Poster" moment

Many Small Panels:
• Rapid-fire sequence
• Montage effect
• Time passing quickly
• Multiple actions

Visual Continuity

📐 Maintaining Consistent Visual Flow

The 180-Degree Rule:
────────────────────────────────────

In film and comics, camera stays on one side of "action line"

Example: Two characters talking

    [Camera Zone]
    
A ------------- B
    
    [Camera Zone]

Camera should stay in top OR bottom zone
Don't cross the line mid-conversation

Why?
• Maintains spatial relationships
• Characters don't "flip" positions
• Reader stays oriented
• Professional look

Breaking the Rule:
• Establishing shot first (shows full space)
• Then can move more freely
• OR intentionally disorient reader

Character Consistency:
────────────────────────────────────

Same Character, Every Panel:
• Proportions stay same
• Costume details consistent  
• Color palette identical
• Distinctive features maintained

Create Reference:
• Character turnaround
• Color swatches
• Detail notes
• Keep visible while working

Props and Background Consistency:
────────────────────────────────────

Scene Continuity:
• Furniture stays in place
• Background elements consistent
• Light direction constant
• Time of day coherent

Continuity Errors to Avoid:
❌ Coffee cup full, then empty, then full
❌ Character's shirt color changes
❌ Sun from left, then right
❌ Background elements disappear

Create Scene Reference:
• Rough map of space
• Note light source
• Mark prop positions
• Check each panel against it

Eye-Line Match:
────────────────────────────────────

When character looks off-panel:
• Next panel shows what they see
• From their perspective
• Creates connection
• Powerful storytelling tool

Example:
Panel 1: Character looks off-panel right
Panel 2: Shows door (what they're seeing)
Reader makes connection automatically

────────────────────────────────────

   • Stop browsing
   • Stand out from others
   • Visual impact

2. Communicate Genre/Content
   • What type of book/comic?
   • Who is it for?
   • What's it about?

3. Evoke Emotion

   • Promise of experience

The Thumbnail Test:
────────────────────────────────────

View cover at thumbnail size (1 inch)
Can you still tell:
